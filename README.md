# Trinath Enterprises uPVC Doors & Windows Solutions Website

This is a premium, high-converting marketing website built for **Trinath Enterprises**, a leading provider of premium uPVC doors and windows supporting projects all over Odisha.

## Features

- **Interactive 3D Visualizer**: A lightweight, high-performance CSS 3D model of a casement door and sliding window allowing customers to preview wood frame laminates, glass tint options, and open/slide animations in real-time.
- **Odisha Weather Protection**: Showcases uPVC structural features tailored for Odisha's coastal region (cyclone resistance up to 250km/h and monsoon-proof rain track sills).
- **Direct WhatsApp Leads**: A contact form and configuration button that formats and redirects project inquiries directly to the company's WhatsApp Business number (`7008859673`).
- **Premium Catalog Gallery**: High-end realistic catalog imagery showcasing uPVC systems in modern villas.
- **Mobile First Optimization**: Fully responsive layout optimized for all device sizes (compact sticky header, responsive grids, slide-out mobile drawer).

## Technology Stack

- **Bundling & Dev Server**: [Vite](https://vite.dev/)
- **Core Architecture**: Semantic HTML5, Vanilla JavaScript (ES Modules)
- **Styling & 3D**: Vanilla CSS3 (CSS 3D Transforms, custom properties, glassmorphic layout)
- **Icons**: Custom SVG icons inline

## Getting Started

To run the development server locally:

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

To compile production bundles:

```bash
npm run build
```
